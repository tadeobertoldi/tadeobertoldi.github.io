Cuerpo principal del contenido

La etiqueta main es en HTML5 un etiqueta semantica. Esto quiere decir que tiene que ver con el significado de los contenido de ella. Define el contenido principal de la pagina. Unicamente puede haber una sola etiqueta como esta en todo el documento.
El contenido escrito aqui dentro no se repite en el resto del sitio(todo lo contrario a los encabezados de pagina, los menues de navegacion y los pie de pagina). Es usado por buscadores para indexar.
Conviene usar aqui textos que ayuden a encontrar la pagina luego de provesos de indexacion de los buscadores.

Seccion etiquetas semanticas estructurales mas importantes:

  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADDRESS

Las etiquetas semanticas estructurales estan relacionadas directamente con el significado del contenido y no con los estilos ni las posiciones dentro del documento.

Reglas para etiquetas semanticas:

  • Los dintintos tipos de contenido se puden asociar a las etiquetas SECTION
  • Las unidades de contenido se asocian a las etiquetas ARTICLE. O sea que un SECTION anida multiples ARTICLES
  • Las etiquetas HEADER Y FOOTER pueden ser anidadas dentro de otras HEADER o FOOTER. Osea un FOOTER podria anidar otro HEADER y FOOTER
  • La etiqueta NAV debe ser unica en el documento y corresponde a los elementos de navegacion del documento
  • Reservar etiqueta h1 para usar dentro de etiquetas top level como main, header o footer
